# 2026-05-10 Git 概念学习

## 讨论内容

- Git 和 GitHub 的区别：Git 是本地版本控制工具，GitHub 是云端代码托管平台
- Git 工作流四区域：工作区 → 暂存区 → 本地仓库 → 远程仓库
- commit 的意义：快照、可回退、可追溯、可协作
- README 的作用：项目入口说明书，给自己也给他人看
- Git 文件找回：不同删除阶段对应不同恢复方法
