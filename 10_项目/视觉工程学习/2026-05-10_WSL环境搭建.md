# 2026-05-10 WSL2 + VS Code 开发环境搭建

## 操作步骤

- 管理员 PowerShell 运行 `wsl --install -d Ubuntu-22.04`，安装 WSL2 内核和 Ubuntu 22.04
- 重启电脑后再次运行，创建用户 `rainz`，设置密码
- WSL 内 `sudo apt update && sudo apt install python3-venv python3-pip -y` 安装 Python 工具链
- 创建项目目录 `~/code/vision-engineering-learning`
- 创建虚拟环境 `python3 -m venv .venv` 并激活
- 运行 `hello_python.py` 验证环境正常
- VS Code 安装 Remote - WSL 扩展，连接 WSL，左下角显示 `WSL: Ubuntu`

## 踩坑记录

- **密码输入不显示**：Linux 安全设计，密码输入时屏幕无任何回显，盲打后回车即可
- **Ctrl+C/V 不能粘贴命令**：传统 cmd 窗口不支持，用右键粘贴，或直接用 VS Code 终端（`Ctrl+`` ` 打开）
- **Powershell 输入反引号导致卡住**：cmd 里误输入 `` ` `` 会让 shell 认为命令未完，按 `Ctrl+C` 取消重来
- **文件名 typo**：`python hello_python.py` 打成了 `heo_python.py`，导致 file not found
