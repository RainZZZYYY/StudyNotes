# 2026-05-10 任务模板双轨制

## 操作步骤

- 发现 `add-task` 的任务模板只适配了学习类项目，不适用截止日期项目
- 设计两套模板，按项目类型自动选用
- 更新了跨项目复制时的字段转换规则

## 两套模板对比

| 字段 | 学习类 | 截止日期类 |
|------|--------|-----------|
| 学习内容 | ✅ | - |
| 理论点 | ✅ | - |
| 任务描述 | - | ✅ |
| 交付物 | - | ✅ |
| 截止日期 | - | ✅ |
| 产出 | ✅ | - |
| 完成标准 | ✅ | ✅ |

跨项目复制时自动转换：理论点 → 任务描述，产出 → 交付物。
