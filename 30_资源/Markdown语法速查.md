# Markdown 语法速查

Markdown 是一种轻量级标记语言，用纯文本语法编写结构化文档，适合写笔记、技术文档、README。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

`#` 后面加空格，数量越少级别越高。

## 文本样式

| 样式 | 语法 |
|------|------|
| 加粗 | `**文本**` |
| 斜体 | `*文本*` |
| 删除线 | `~~文本~~` |
| 行内代码 | `` `代码` `` |

## 列表

```markdown
- 无序列表
1. 有序列表
  - 嵌套缩进两个空格
```

## 链接与图片

```markdown
[链接文本](https://example.com)
![替代文本](image.png)
```

## 代码块

````markdown
```python
print("hello")
```
````

## 引用

```markdown
> 引用内容
>> 嵌套引用
```

## 表格

```markdown
| 列1 | 列2 |
| --- | --- |
| A   | B   |
```

对齐：`:---` 左，`:---:` 中，`---:` 右。

## 任务列表

```markdown
- [x] 已完成
- [ ] 未完成
```

## 分割线

```markdown
---
```

## 转义

用 `\` 转义特殊符号：`\*`、`\#`。

## 扩展语法

脚注：`[^1]` + `[^1]: 内容`
行内公式：`$E=mc^2$`
块级公式：`$$a^2+b^2=c^2$$`

## 易错点

- `#` 后忘加空格 → `# 标题` 才对
- 代码块未闭合 → 后续内容被吞
- 表格竖线对齐可提高源码可读性
